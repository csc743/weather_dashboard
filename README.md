# weather_dashboard
Playful Louisville Weather Dashboard 🌤️

A simple, playful weather dashboard built with Python, Dash, and Plotly.  
It shows data about daily high temperatures, daily averages, and change over the day 

## Features
- Hourly temperature trend
- Daily average temperature bar chart
- Highest tempature recorded card

Requirements:
dash
plotly
pandas
sqlalchemy
python-dotenv
